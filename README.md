# Conference Helper (학회 도우미)

국제 학회에서 실시간 음성인식과 번역 자막을 제공하는 모바일 웹앱입니다.

## 주요 기능
- **실시간 음성인식**: Web Speech API를 사용하여 말소리를 즉시 텍스트로 변환합니다.
- **다국어 번역**: Google Translate API를 통해 원하는 언어로 실시간 번역합니다.
- **카메라 오버레이**: 카메라 화면 위에 번역된 자막이 표시되어 발표를 보며 내용을 파악할 수 있습니다.
- **캡처 및 갤러리**: 자막이 포함된 화면을 스크린샷으로 저장하고 나중에 다시 볼 수 있습니다.
- **기록 저장**: 전체 번역 기록을 텍스트 파일로 다운로드할 수 있습니다.
- **PWA 지원**: 홈 화면에 추가하여 앱처럼 사용할 수 있습니다.

## 기술 스택
- HTML5, CSS3, Vanilla JavaScript
- Web Speech API
- Google Translate API (무료 엔드포인트)
- Canvas API (이미지 캡처)

## 사용 방법
1. 브라우저에서 `index.html`을 엽니다 (HTTPS 환경 권장).
2. 인식 언어와 번역 언어를 선택합니다.
3. '시작하기' 버튼을 누르고 마이크 및 카메라 권한을 승인합니다.
4. 인식된 텍스트와 번역된 자막이 실시간으로 화면에 표시됩니다.
5. 중간에 중요한 장면은 캡처 버튼으로 저장할 수 있습니다.

## 주의 사항
- **브라우저**: Chrome 또는 Edge 브라우저에서 가장 잘 작동합니다.
- **HTTPS**: 카메라와 마이크 권한은 보안 연결(HTTPS) 또는 localhost에서만 작동합니다.
- **네트워크**: 번역 기능을 위해 인터넷 연결이 필요합니다.
